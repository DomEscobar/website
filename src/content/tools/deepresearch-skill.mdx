Dieser Skill ist für Recherchen gedacht, bei denen ein schnelles Summary nicht reicht.

Er verwendet einen festen Ablauf:

```txt
plan -> discover -> fetch/read -> extract evidence -> cross-check -> synthesize -> cite
```

## Wann benutzen?

- Tool- oder Technologievergleiche
- Markt- und Wettbewerbsrecherche
- Implementierungsrecherche
- Entscheidungen, bei denen Quellenqualität wichtig ist
- Fragen wie: "Ist das nur Googeln?"

## Was der Skill erzwingt

- ein Recherche-Plan-Artefakt
- eine Quellenliste
- extrahierte Evidenz mit URLs
- Confidence-Notizen
- einen zitierfähigen Report
- eine Validierung der Artefakte vor dem Ergebnis

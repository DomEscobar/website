Dieser Skill ist kein normaler Code-Review-Skill.

Er prüft Änderungen, die beeinflussen, wie zukünftige Agents arbeiten:

- `AGENTS.md`, `TOOLS.md`, `SOUL.md`, `USER.md`, `HEARTBEAT.md`
- Skill-Dateien und Skill-Support-Dateien
- Runtime-Konfigurationen
- Hooks, Allowlisten, Tooling, Sandbox- oder Prompt-Injection-Änderungen

## Output

Der Skill gibt zuerst ein JSON-Urteil aus:

```json
{
  "verdict": "ship|ship-with-notes|needs-fix|unsafe",
  "confidence": "high|medium|low",
  "blockers": []
}
```

Danach folgt eine kurze menschliche Zusammenfassung.
